
# ⚙️ Módulo 8: Técnicas de Post-Explotación

## 📖 Resumen

Se enfoca en las acciones después de comprometer un sistema.

## 🧠 Análisis

El acceso inicial no representa el mayor riesgo, sino lo que el atacante puede hacer después.

## 🔍 Técnicas principales

* Escalada de privilegios
* Movimiento lateral
* Persistencia
* Recolección de datos

## 🛠️ Aplicación

* Mantener acceso al sistema comprometido
* Evadir mecanismos de detección
* Extraer credenciales y datos sensibles

## ⚠️ Conclusión

Una pequeña vulnerabilidad puede escalar a un compromiso total del sistema.
